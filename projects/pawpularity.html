Cloud Based RAG Chatbot with Streamlit and AWS Bedrock Knowledge Base

  • Tech Stack: Python, AWS Bedrock, Amazon Claude 3.5, Bedrock Knowledge Base, Streamlit
  • Website URL: Link
  • Github URL: Project Link

This project developed a Streamlit-powered RAG chatbot leveraging AWS Bedrock Knowledge Base and Claude 3.5 Haiku to provide document-grounded answers from enterprise filings like Amazon's 10-K.

It demonstrates a robust solution for interactive, explainable insights from company knowledge, crucial for financial analysis and compliance.
